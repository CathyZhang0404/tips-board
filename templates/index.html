Manual tip splits (optional)

Use this for party / special tips: choose a tipped payment from today’s Clover data, then enter each person’s share (any positive numbers — only the ratios matter). Examples: three people each type 1 → each gets ⅓; two people type 1 → 50/50; 1 and 2 → ⅓ and ⅔. We normalize shares to 100%. Those payments ignore shift times; every other tipped payment still uses time + equal split among who was working.

Refresh Clover data above first — then add a rule and pick a payment from the list.

Email not configured

Set SMTP variables in CLOVER_Tips/.env or tip_dashboard/.env, then restart the server (Ctrl+C, then uvicorn again).

Gmail: SMTP_HOST=smtp.gmail.com, SMTP_PORT=587, your address as SMTP_USERNAME / SMTP_FROM_EMAIL, and an App password as SMTP_PASSWORD (not your normal Gmail password if 2FA is on).
